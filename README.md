clang -std=c2x -O3 -Wall -Wextra -I. -o review_siftr2_log review_siftr2_log.c  
%  
  
compile in MacOS  
% make  
clang -std=c2x -O3 -Wall -Wextra -I. -o review_siftr2_log review_siftr2_log.c  
%
