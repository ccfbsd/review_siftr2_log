# review_siftr2_log  
This is the C code to parse and analyze logs generated by the custom SIFTR 2.x  
tool that still uses the ALQ(9) for logging.  
  
compile examples:  
  
compile in FreeBSD  
% gmake  
clang -std=c23 -O3 -Wall -Wextra -pthread -I. -o review_siftr2_log review_siftr2_log.c  
%  
  
compile in MacOS  
% make  
clang -std=c23 -O3 -Wall -Wextra -pthread -I. -o review_siftr2_log review_siftr2_log.c  
%
